# 2D Minecraft - Made with pygame
This is a small game project, where you can play a 2D junky version of Minecraft with Python!

## Controls
- `A`, `D` or arrows - movement
- `Space`, `W` or `Up` arrow - jump
- `1`, `2`, `3`, ... - switch between items
- `Left` click - destroy
- `Right` click - place
- `F1` - debug
- `F2` - show FPS
